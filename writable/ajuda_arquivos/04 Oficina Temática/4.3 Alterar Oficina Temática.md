# Alterar Oficina Temática

Use esta tela para atualizar dados de uma oficina.

---

## O que pode ser alterado
- Nome e descrição.
- Arquivos, materiais e recursos.

## Regras importantes
- Os arquivos e materiais são atualizados pela lista atual.
- Se você remover um arquivo ou foto da lista, ele é excluído do sistema.
- Ao adicionar novo arquivo, é obrigatório anexar o arquivo.
- Ao adicionar novo material, é obrigatório anexar a foto.

## Como fazer
1. Ajuste os campos principais.
2. Revise a lista de arquivos e materiais.
3. Atualize os recursos de trabalho.
4. Clique em Salvar.
