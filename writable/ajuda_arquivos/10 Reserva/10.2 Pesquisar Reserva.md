# Pesquisar Reserva

Use para localizar reservas cadastradas.

---

## Filtros disponíveis
- Data da reserva de / até.
- Data de cadastro de / até.
- Hora início e hora fim.
- Tipo de reserva.
- Status.
- Turma escolar.
- Nome da escola.
- Número de convidados de / até.

## Regras importantes
- Preencha somente os filtros desejados.

## Resultado
- A lista mostra as reservas encontradas.
- Você pode abrir a reserva para alterar.
