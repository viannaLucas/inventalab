# Alterar Usuário

Use esta tela para atualizar dados e permissões de um usuário.

---

## Regras importantes
- O login não pode ser alterado.
- A senha não é alterada nesta tela.
- Não é permitido desativar o próprio usuário.
- Se você remover a marcação de ativo, o usuário será desativado.

## Como fazer
1. Ajuste nome, foto e permissões.
2. Defina se o usuário está ativo.
3. Clique em Salvar.
