# Alterar Recurso de Trabalho

Use esta tela para atualizar ferramentas e equipamentos.

---

## O que pode ser alterado
- Dados principais do recurso.
- Foto.
- Garantias relacionadas.

## Regras importantes
- Se o tipo for Equipamento, Requer habilidade e Uso exclusivo são obrigatórios.
- Ao enviar nova foto, a antiga é substituída.
- A lista de garantias é substituída pela lista atual.

## Como fazer
1. Ajuste os campos necessários.
2. Atualize a foto se for preciso.
3. Revise as garantias.
4. Clique em Salvar.
