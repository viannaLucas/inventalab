# Pesquisar Recurso de Trabalho

Use para localizar recursos cadastrados.

---

## Filtros disponíveis
- Nome.
- Marca do fabricante.
- Tipo.
- Requer habilidade.
- Uso exclusivo.
- Situação do recurso.
- Quantidade disponível de / até.

## Regras importantes
- Preencha somente os filtros desejados.

## Resultado
- A lista permite abrir o recurso para alterar.
