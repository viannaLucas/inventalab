# Pesquisar Participante

Use para localizar participantes cadastrados.

---

## Filtros disponíveis
- Nome.
- E-mail.
- CPF.
- Cidade.
- CEP.
- Suspenso (Sim ou Não).

## Regras importantes
- Preencha somente o que deseja filtrar.

## Resultado
- A lista permite abrir o participante para alterar.
