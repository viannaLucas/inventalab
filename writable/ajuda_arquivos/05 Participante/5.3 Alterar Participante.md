# Alterar Participante

Use para atualizar dados do participante.

---

## O que pode ser alterado
- Dados pessoais e endereço.
- Situação de suspensão.
- Termo de responsabilidade.
- Observações e habilidades.

## Regras importantes
- E-mail não pode ser alterado.
- A senha não é alterada nesta tela.
- Se marcar cobrança no responsável, o nome deve ter pelo menos 5 caracteres.
- Ao enviar um novo termo, o arquivo antigo é substituído.
- Se marcar para excluir o termo, ele é removido do sistema.

## Como fazer
1. Ajuste os dados necessários.
2. Atualize ou remova o termo, se preciso.
3. Clique em Salvar.
