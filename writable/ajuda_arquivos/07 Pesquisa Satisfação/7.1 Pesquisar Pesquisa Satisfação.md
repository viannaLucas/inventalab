# Pesquisar Pesquisa Satisfação

Use para localizar respostas da pesquisa de satisfação.

---

## Filtros disponíveis
- Uso do espaço (nota de / até).
- Atendimento (nota de / até).
- Equipamentos (nota de / até).
- Data de resposta (de / até).

## Regras importantes
- Preencha somente os filtros desejados.

## Resultado
- A lista mostra as respostas encontradas.
- Você pode abrir cada resposta para visualizar os detalhes.
