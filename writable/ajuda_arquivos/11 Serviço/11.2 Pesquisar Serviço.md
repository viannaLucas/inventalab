# Pesquisar Serviço

Use para localizar serviços cadastrados.

---

## Filtros disponíveis
- Nome.
- Descrição.
- Unidade.
- Ativo.
- Valor de / até.
- Código.

## Regras importantes
- Preencha somente os filtros desejados.

## Resultado
- A lista permite abrir o serviço para alterar.
