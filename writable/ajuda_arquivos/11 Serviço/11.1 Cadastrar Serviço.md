# Cadastrar Serviço

Use esta tela para registrar serviços oferecidos pelo espaço.

---

## Campos obrigatórios
- Nome.
- Descrição.
- Valor: maior que zero.
- Unidade.
- Ativo: Sim ou Não.
- Código do serviço: obrigatório e único.

## Produtos vinculados
Alguns serviços podem ter **insumos vinculados**. Por exemplo, o uso da **Impressora 3D** pode ter o produto **filamento** vinculado, e o serviço de uso da **lixadeira** pode ter o produto **lixa** vinculado. Isso permite controlar o estoque dos insumos, apoiando a reposição com base no **estoque mínimo** do produto.

É possível vincular **mais de um produto** ao mesmo serviço e definir a **quantidade** de cada item. Quando o serviço é selecionado em uma reserva, a quantidade do produto vinculado é **baixada automaticamente do estoque**.

Para adicionar um produto ao serviço, utilize o formulário na área **Lista de Produtos do Serviço**. Clique na **lupa** para selecionar o produto, defina a quantidade e pressione o botão azul com o ícone **+**.

![](servico_produto_formulario.png){max-width=480 center}

Para excluir um produto vinculado, utilize o botão vermelho com ícone de **lixeira** na linha do produto.

![](servico_produto_formulario_excluir.png){max-width=480 center}

## Dados Fiscais
Todo serviço cadastrado no sistema deve estar vinculado a um serviço cadastrado no **MXM do SESC**. Essa vinculação é **1 para 1**: um serviço do sistema pode estar ligado a apenas **um** serviço do MXM, e um serviço do MXM não pode ser vinculado a mais de um serviço no sistema.

![](servico_dados_fiscais.png){max-width=480 center}

A vinculação é feita pelo campo **Código Serviço Cadastrado no MXM**. Esse código deve ser obtido junto ao departamento responsável do SESC. Outro campo obrigatório é o **Código do Tipo de Operação**.

Ao preencher o código do MXM, o sistema realiza uma consulta e preenche os demais dados fiscais automaticamente. Quando o código é válido, o nome do serviço correspondente no MXM aparece logo abaixo do campo.

![](servico_dados_fiscias_campo_codigo.png){max-width=480 center}

## Regras importantes
- O código não pode se repetir.
- Quando o serviço é usado em cobranças, os produtos vinculados baixam do estoque.

## Como fazer
1. Preencha os dados do serviço.
2. Informe o código.
3. Vincule produtos se necessário.
4. Clique em Cadastrar.
