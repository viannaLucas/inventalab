# Alterar Serviço

Use esta tela para atualizar informações do serviço.

---

## Regras importantes
- O código do serviço é obrigatório e deve ser único.
- Valor precisa ser maior que zero.
- A lista de produtos vinculados é substituída pela lista atual.

## Como fazer
1. Ajuste os campos necessários.
2. Revise o código.
3. Atualize os produtos vinculados.
4. Clique em Salvar.
