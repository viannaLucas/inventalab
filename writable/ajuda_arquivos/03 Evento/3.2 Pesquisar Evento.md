# Pesquisar Evento

Use para localizar eventos cadastrados.

---

## Filtros disponíveis
- Nome.
- Vagas limitadas.
- Inscrições abertas.
- Divulgar.
- Número de vagas de / até.
- Data de início de / até.

## Resultado
- A lista mostra os eventos encontrados.
- A partir dela você pode abrir o evento para alterar.

## Regras importantes
- Preencha somente os filtros que deseja usar.
- Datas inválidas não são aceitas.
