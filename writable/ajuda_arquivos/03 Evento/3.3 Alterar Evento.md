# Alterar Evento

Use esta tela para atualizar informações de um evento. Além dos dados principais, aqui ficam disponíveis os comandos para **controle de presença**, **impressão de listas** (presença e retirada de material) e **exportação de participantes**.

---

## O que pode ser alterado
- Dados principais do evento.
- Vagas, inscrições e divulgação.
- Reservas do espaço.
- Participantes e listas de presença.
- Imagem do evento.

## Controle de presença

![](evento_botoes_presenca.png){max-width=90% center}

Se o evento precisar de acompanhamento de presença, é possível cadastrar **listas de presença**. Isso é útil quando o evento ocorre em **turnos** (manhã, tarde) ou em **dias diferentes** (primeiro dia, segundo dia). Nesse caso, você pode criar **mais de uma lista**, uma para cada período.

![](evento_cadastro_controle_presenca.png){max-width=90% center}


O sistema solicita qual lista deseja utilizar e, após selecionar, exibe a lista de participantes. 
Ao lado de cada participante existe o botão **Adicionar Presença** para marcar quem compareceu.

![](evento_selecionar_lista_presenca.png){max-height=300 max-width=90% center}

Para registrar a presença, clique em **Definir Presença**. 

![](evento_lista_presenca_botao.png){max-height=300 max-width=90% center}

## Presença impressa
Se preferir controlar a presença em papel, utilize o botão **Imprimir Lista Presença**. 

![](evento_lista_presenca_botao_imprimir.png){max-width=180 center}

O sistema pedirá qual lista será impressa e abrirá a versão de impressão no navegador.

![](evento_lista_presenca_impressao.png){max-height=250 max-width=90% center}

## Lista de retirada de material

![](evento_lista_entrega_material_botao.png){max-width=180 center}

Alguns eventos exigem o controle de entrega de materiais. Para isso, use o botão **Imprimir Lista Material** para gerar a lista física de retirada.

![](evento_lista_entrega_material_impressao.png){max-height=250 max-width=90% center}

## Exportação de participantes
Quando for necessário obter os dados de contato dos participantes, use o botão **Exportar Participantes**. O sistema fará o download de um arquivo com a lista de participantes.

![](evento_lista_exportar_participantes_botao.png){max-width=180 center}

## Lista de participantes
Cada evento possui sua própria lista de participantes. É aqui que você adiciona, acompanha e gerencia quem está inscrito no evento.

![](evento_lista_participantes.png){max-height=400 max-width=90% center}

Para adicionar um participante, clique na **lupa** do campo **Participante**, selecione a pessoa desejada e, em seguida, pressione o botão azul com o símbolo **+**. Depois de incluir todos os participantes necessários, finalize clicando no botão **Alterar** no final da página para salvar as mudanças.

![](evento_lista_participante_formulario.png){max-width=380 center}

Quando um participante ainda não possui pagamento definido como **Pago**, a linha aparece com o botão de **excluir** (lixeira). Esse botão permite remover o participante da lista.

![](evento_lista_participante_sem_pagamento.png){max-width=420 center}

Só é possível excluir participantes que **não possuem pagamento** marcado como **Pago**.

Ao cadastrar um novo participante, nenhum pagamento é criado automaticamente. Para vincular o pagamento, use o botão **amarelo** com o símbolo de **dinheiro/moeda** e defina a situação como **Pago**. Após confirmar, o registro de pagamento é criado, o ícone fica **verde** e o botão de exclusão deixa de aparecer.

![](evento_lista_participante_com_pagamento.png){max-width=480 center}

## Regras importantes
- Se vagas limitadas estiver ativo, o número de vagas deve ser maior que zero.
- O total de participantes não pode ultrapassar o número de vagas.
- Reservas do espaço passam por verificação de conflito. Se houver sobreposição com outras reservas, a alteração é bloqueada.

## Como fazer
1. Ajuste os campos necessários.
2. Revise as reservas do espaço.
3. Revise a lista de participantes e presença.
4. Clique em Salvar.
