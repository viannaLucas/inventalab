# Relatório Reservas

Use para acompanhar o volume de reservas e seus dados principais.

---

## O que o relatório mostra
- Reservas por período.
- Informações de data, horário, tipo e quantidade de pessoas.

## Filtros
- Data início.
- Data fim.
- Agrupamento: dia, semana ou mês.

## Regras importantes
- Se as datas forem inválidas, o sistema ajusta para um período padrão.
- Se a data inicial for maior que a final, o sistema corrige automaticamente.

## Como fazer
1. Informe data início e data fim.
2. Escolha o agrupamento.
3. Clique em Gerar relatório.
