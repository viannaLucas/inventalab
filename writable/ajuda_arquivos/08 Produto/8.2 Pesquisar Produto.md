# Pesquisar Produto

Use para localizar produtos cadastrados e acompanhar o estoque.

---

## Filtros disponíveis
- Nome.
- Estoque mínimo de / até.
- Estoque atual de / até.
- Ativo (Sim ou Não).
- Valor de / até, quando disponível.
- Abaixo do estoque mínimo (Sim ou Não).

## Regras importantes
- Preencha somente os filtros desejados.

## Resultado
- A lista mostra os produtos encontrados.
- Você pode abrir cada produto para alterar.
