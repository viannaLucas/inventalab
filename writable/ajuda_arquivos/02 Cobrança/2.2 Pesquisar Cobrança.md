# Pesquisar Cobrança

Use esta tela para localizar cobranças já registradas e acessar seus detalhes.

---

## Filtros disponíveis
- Participante.
- Situação.
- Valor de / até.
- Data de / até.

## Regras importantes
- Preencha somente os filtros que deseja usar.
- Datas inválidas não são aceitas.

## Resultado da pesquisa
- A lista mostra ID, participante, data, valor e situação.
- Você pode abrir para alterar ou excluir, conforme sua permissão.
- Se a cobrança estiver paga, a alteração é bloqueada pelo sistema.

## Como fazer
1. Informe os filtros desejados.
2. Clique em Pesquisar.
3. Use os botões da lista para abrir ou excluir.
