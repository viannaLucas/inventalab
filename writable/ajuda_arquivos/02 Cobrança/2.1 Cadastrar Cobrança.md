# Cadastrar Cobrança

Use esta tela para registrar a cobrança de serviços usados por um participante.

---

## Para que serve
- Registrar valores de serviços.
- Guardar a situação da cobrança.

## O que precisa informar
- Participante: obrigatório. Selecione pelo botão de busca.
- Data: obrigatória.
- Situação: escolha entre Aberta, Paga ou Cancelada.
- Observações: obrigatório.

## Serviços da cobrança
- Adicione pelo menos um serviço.
- Quantidade deve ser número inteiro maior que zero.
- Valor unitário deve ser maior que zero.
- O total é calculado automaticamente com base nos serviços informados.

## Regras importantes
- Sem participante, data, situação ou observações o sistema não salva.
- Se o total ficar zero, a cobrança não é aceita.
- Quando um serviço consome produtos do estoque, o sistema bloqueia se não houver quantidade suficiente.

## Como fazer
1. Clique em buscar participante e selecione.
2. Informe a data e a situação.
3. Escreva as observações.
4. Adicione os serviços e as quantidades.
5. Clique em Cadastrar.
